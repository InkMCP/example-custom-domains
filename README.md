# Custom Domains

A simple static site demonstrating custom domain setup with Ink, ready to deploy.

## Setup

Open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```

## Custom Domain Setup

After deploying with Ink, you can add a custom domain:

1. Go to your service settings on [ml.ink](https://ml.ink)
2. Add your custom domain
3. Configure DNS records as shown in the app
4. SSL is provisioned automatically

## Deploy with Ink

Connect the [Ink MCP server](https://docs.ml.ink/quick-start) to your AI agent and prompt:

> Deploy this static site with Ink and add a custom domain.
